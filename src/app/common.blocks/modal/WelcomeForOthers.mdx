PATH.JERRYIO is a multi-purpose path editor/planner which can be used to design routes for one-minute driver skill and generate path files. It is a Progressive Web App (PWA) that is installable and can be used on any device without an internet connection.

![A banner showing PATH.JERRYIO running on the iMac, MacBook, iPad, and iPhone](static/readme-banner-0.5x.png)

More information about this editor can be found on the [GitHub Page](https://github.com/Jerrylum/path.jerryio).
